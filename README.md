## How to Use

Simply open `index.html` in a web browser to view the portfolio. No build process or dependencies required!

```bash
# Clone the repository
git clone https://github.com/NANDYALA-NEELAKANTESWAR-REDDY/Storynarrationportfolio.git

# Navigate to the directory
cd Storynarrationportfolio

# Open index.html in your browser
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

## Customization

To personalize this portfolio:

1. Update the content in `index.html`:
   - Replace placeholder text with your own story
   - Update project descriptions
   - Add your skills and technologies
   - Change contact information and social links

2. Modify colors in `style.css`:
   - Edit CSS variables in `:root` for global color scheme
   - Customize gradient backgrounds
   - Adjust fonts to match your style

3. Enhance interactions in `script.js`:
   - Modify animation speeds and effects
   - Add new interactive features
   - Customize scroll behaviors
